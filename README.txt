# ToonTanks
Assets, blueprints, and C++ code from ToonTanks Unreal Engine project

ToonTanks is a project created using Unreal Engine 5. This project contains
a custom map with mutiple C++ components combined with Unreal's
Blueprint Visual Scripting. Gameplay consists of simple game with a
player-controlled tank and AI-controlled enemy turrets. The player has 2 lives
and must destroy all enemy turrets to win. Visual effects and sound are also
featured in this project and provide a smoother experience.
